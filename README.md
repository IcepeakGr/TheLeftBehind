# The Left Behind

A custom Zombie Apocalypse server project for Java Minecraft. It is designed as a persistent, open-world survival game inspired by *The Walking Dead*, focusing completely on custom gameplay systems built from scratch.


## ✨ Custom Features
*   **Survivor Vitals:** Tracks player dehydration, blood loss, and virus infection levels over time via background task processors.
*   **Horde Attraction Engine:** Fully integrated gun mechanics that generate audio vectors, causing custom NMS zombies to drop their current actions and hunt down the noise location.
*   **Custom NMS Zombies:** Overrides vanilla Minecraft pathfinding goals to optimize server performance while creating smart, fast-moving threat mechanics.
